# Crop Price Prediction

## Description
This project predicts crop prices based on agricultural factors using regression techniques.

## Dataset
Crop production and price dataset from Kaggle.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Model
- Linear Regression

## Steps Performed
- Data cleaning and preprocessing
- Selection of numerical features
- Train-test split
- Model training using Linear Regression
- Evaluation using MAE and MSE

## Result
The model predicted crop prices with reasonable accuracy and acceptable error values.
